/*title: Contrato de alquilerdescription: Contrato de alquiler - El Proceso De Alquiler*/Para cada alquiler, Rennty proporciona un contrato de alquiler ya completado.Este contrato está disponible en [Tus Alquileres](/es/browse/rentals) una vez se ha reservado y pagado el alquiler en la web.Si en el contrato figuran datos incorrectos (número del carnet de conducir incorrecto, error en la fecha de expedición del carnet, etc) es posible modificar esta información en el contrato móvil, desde la app:La página del [contrato de alquiler](/es/contrato) detalla las cláusulas del mismo.
